# 高级用法：自定义字体 / 字号 / 修改译文

面向想自行调整汉化包的高级用户。所有工具在 `tools\`，需要 Python 3 + `pip install pillow numpy`，命令建议用 `py -X utf8` 运行避免控制台编码报错。

## 1. 更换字体

游戏全部界面/字幕只用一个字体包 `GameData\Textures\orbitfonts.utx`，用 `font_gen.py` 可以从任意 TTF/OTF 重新生成：

```
py -X utf8 tools/make_charset.py charset.txt
py -X utf8 tools/font_gen.py --font 你的字体.ttf --charset charset.txt --out orbitfonts.utx
```

生成后覆盖 `GameData\Textures\orbitfonts.utx` 即可，无需其他改动。

对字体的要求：
- **必须覆盖你所用文本的全部汉字**（推荐至少 GB2312 范围）。缺字会显示为空白（回落到空白字形）
- 授权允许再分发才可以打包发布（思源黑体/更纱黑体等 OFL 字体可以；**微软雅黑等 Windows 自带字体不可以**）
- 粗体字形在小字号（13px 行高）下更清晰，原版风格也是 Bold

## 2. 调整字号

游戏按用途固定使用 5 个字号对象（OrbitBold8/12/15/18/24），字号↔字形行高（像素）的默认映射取自官方日文版：

| 字号对象 | 默认行高 | 主要用途 |
|---|---|---|
| OrbitBold8 | 13 | 菜单按钮、小字 UI |
| OrbitBold12 | 16 | 一般 UI |
| OrbitBold15 | 20 | 字幕 |
| OrbitBold18 | 24 | 标题类 |
| OrbitBold24 | 29 | 大标题 |

想整体放大缩小某档，用 `--cells` 覆盖（只写想改的档位）：

```
py -X utf8 tools/font_gen.py --font 字体.ttf --charset charset.txt --cells "15=22,8=14" --out orbitfonts.utx
```

注意：UI 布局按行高排版，行高改动过大会导致文字挤压或溢出边框；建议每档 ±2px 内微调。

## 3. 修改译文

文本分两类，都必须以 **GBK 编码、无 BOM** 保存（用记事本"ANSI"或编辑器选 GBK；**不要存成 UTF-8**，会显示乱码）：

### 3.1 字幕 / 提示 / 关卡文本（`GameData\System\*.int`）

普通 ini 风格键值，直接编辑等号右边的值即可，例如 `subtitles_geo_01.int`：

```ini
[SubtitleEvents]
GEO01_Boss_066=博斯：中文字幕内容
```

注意保留 `%k %o %s` 之类的占位符（运行时替换为按键名等）。

### 3.2 菜单文字（`GameData\System\PropertyOverrides\*.txt`）

菜单文字编译在代码包里，通过 Fix 的 PropertyOverrides 机制覆盖：文件名必须是 `包名.类名.txt`（如 `XInterfaceCTMenus.CTMenuMain.txt`），内容是 defaultproperties 风格的整行覆盖：

```
MenuOptions(0)=(Blurred=(Text="新的战役",PosX=0.68375,PosY=0.48),BackgroundBlurred=(PosX=0.68375,PosY=0.48,ScaleX=0.38,ScaleY=0.04333),OnSelect="NewGameSelected",Style="ButtonTextStyle1")
```

⚠ 必须整个结构体一起写（从 `reference\export\` 里对应类的 .uc 源码复制原值，只改 `Text=`/`HelpText=` 字段），只写 Text 会丢掉坐标等其他字段。加载失败会静默跳过——改完进游戏看不到效果，先检查文件名与结构体拼写。

### 3.3 用了新汉字之后

译文里出现字体包没有的字会显示空白。改完译文建议重跑字符集扫描+字体生成（发布版字符集就是从全部译文扫描出来的）；开发期偷懒可以直接用 `make_charset.py` 的 GB2312 全集，覆盖绝大多数简体用字。

## 4. 查看 / 反解字体包（调试用）

```
py -X utf8 tools/swrc_package.py orbitfonts.utx                     # 导出表概览
py -X utf8 tools/swrc_package.py orbitfonts.utx font OrbitBold15   # 字形数/页数/键分布
py -X utf8 tools/swrc_package.py orbitfonts.utx tex OrbitBold15_Page00 页.png  # 贴图页转 PNG
```

## 5. 依赖关系提醒

- CJK 能显示的前提是 **Fix（≥ 含 CJK 渲染支持的版本）的 Mod.dll**——字体和文本只是数据，双字节配对逻辑在 Mod.dll 里
- Steam"验证游戏完整性"会还原被覆盖的原版文件，验证后需重装汉化包
